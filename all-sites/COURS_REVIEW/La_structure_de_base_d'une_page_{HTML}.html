Structure minimale d'une page en HTML

Pour qu?une page HTML soit déclarée valide, elle doit obligatoirement comporter certains éléments et suivre un schéma précis.
Voici donc ci-dessous le code minimum pour créer une page HTML valide.

- Le DOCTYPE : il indique que la page est codée en HTML

- Le html : il va contenir tout ce qui sera présent dans la page

- Le head : il contient toute les meta informations relatives à la page

- Le body : il contient tout le contenu "visible" de la page : les textes, images, liens, vidéos, ect...

- L'élément meta : et son attribut "charset" qui va nous permettre de définir l'encodage de la page.
Nous autres, Français, nous utiliserons ka valeur "utf-8"
