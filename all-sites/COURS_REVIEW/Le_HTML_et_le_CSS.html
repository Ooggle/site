HTML est l'abréviation de "HyperText Markup Language" soit en français "langage de balisage hypertexte".
Le HTML a pour fonction de structurer et de donner du sens a du contenu. Il peut par exemple permettre d'insérer différents type d'éléments comme du teste, des liens (interne ou externe), des images, ect.

Le CSS est le iminutif de Cascading StyleSheets, ou feuilles de styles en cascade.
Le CSS a pour rôle de mettre en forme fu contenu en lui appliquant ce que l'on appelle des styles. On va par exemple pouvoir définir la taille, la couleur ou l'alignement d'un texte.

On va donc utiliser le CSS sur le code HTML afin d'enjoliver le résultat visuel final.
